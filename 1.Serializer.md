# Serializer

##### Serializers

쿼리셋, 모델 인스턴스 등의 복잡한 데이터를 JSON, XML 등의 컨텐트 타입으로 쉽게 변환 가능한 python data type으로 변환 시켜줌



##### Deserializers

받은 데이터를 validating 한 후에 parsed data를 complex type으로 다시 변환

데이터를 deserializer 하기 전에 항상 `is_valid()` 함수를 호출해야 한다.
